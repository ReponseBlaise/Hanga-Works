## Hanga Works Frontend

This package contains the Vite + React client for Hanga Works.

### What It Covers

- Public marketing pages and role-aware authentication flows.
- Learner, employer, mentor, institution, and admin navigation.
- Job marketplace browsing, filtering, saving, and application entry points.
- Profile management, notifications, mentorship, certifications, and dashboard screens.

### Local Development

1. Install dependencies with `npm install` from the `frontend/` directory.
2. Start the dev server with `npm run dev`.
3. Use the configured API proxy to reach the backend during development.
